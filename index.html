OPC-UA

Professionally developed, high-performance, open-source, OPC-UA Client and Server SDKs.

Save Time

Leave the OPC-UA implementation details to the experts. Using a toolkit lets you focus on building your application insead rather than interoperability details.

Save Money

Available as open-source or with commercial friendly licensing options at a significantly lower cost than an in-house implementation.

Platform Netural

Implemented in Java to take OPC-UA's platform-neutral approach to the next level. Interoperability with any OPC-UA product on any OS.

Safe and Secure

Safe and secure by default; supports Basic128Rsa15, Basic256, and Basic256Sha256 security profiles.
